A simple example of using Webdriverio with Tauri v2 using vanilla JS and NPM.

## Requirements

- [Tauri prerequisites](https://tauri.app/start/prerequisites/)
- [System WebDriver](https://tauri.app/develop/tests/webdriver/#system-dependencies)

# Setup
```
git clone https://github.com/trymeouteh/tauri-webdriverio-js-npm-example
cd tauri-webdriverio-js-npm-example
npm i
npm run webdriverio
```
